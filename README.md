# J

Static no-build scaffold with a JSON content source.

## Files

- `index.html`
- `admin.html`
- `styles.css`
- `script.js`
- `admin.js`
- `content.json`
- `package.json`
- `images/photo1.jpg` ... `images/photo12.jpg`

## Run locally

```bash
npm run dev
```

This uses:

```bash
npx http-server -c-1 . -p 8080
```
